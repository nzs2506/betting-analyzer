# Betting Analyzer

Static betting anomaly report generated by GitHub Actions and published with GitHub Pages.

## Setup

1. Create a GitHub repository and push these files.
2. In the repository, open `Settings -> Secrets and variables -> Actions`.
3. Add repository secrets:
   - `ODDS_API_KEY`
   - `API_FOOTBALL_KEY` optional
4. Open `Settings -> Pages`.
5. Set source to `Deploy from a branch`.
6. Select branch `main` and folder `/docs`.
7. Run `Actions -> Update Betting Report -> Run workflow`.

The report will be available at:

```text
https://YOUR_USERNAME.github.io/YOUR_REPOSITORY/
```

The scheduled workflow runs once per hour.
